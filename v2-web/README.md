# v2-web

Vue 3 + TypeScript + Vite + Pinia + Vue Router + Element Plus frontend for Module Replacement Project Manager V2.0.

## Run

```bash
npm install
npm run dev
```

The dev server defaults to `http://localhost:5173` and proxies `/api` to `http://127.0.0.1:8000`.

## Static Pages

- `/login`
- `/dashboard`
- `/projects`
- `/checklists`
- `/tasks`
- `/review/:groupId`
