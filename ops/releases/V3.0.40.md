## User Request

- 图片条码准确率改为资料组整体判断：一组照片内只要累计识别并匹配表号、模块号、采集器号三项，即判定资料组通过。
- 给管理员一个弹窗式 list，列出识别不出来的资料组，便于人工判断。

## Changed Scope

- 新增资料组级条码聚合规则，不再要求单张指定照片必须识别出指定条码。
- 保留旧 `photo_accuracy_*` 字段，新增 `group_barcode_accuracy_*` 字段。
- 新增管理员接口 `/local-test/photo-barcode/review-groups`，可查看无法识别和异常不匹配资料组。
- 项目驾驶舱“资料组条码准确率”改为管理员可点击入口，弹窗显示表号、模块号、采集器号、安装人员、缺失项、已识别值、异常值和照片缩略图。
- 弹窗图片使用带鉴权的 blob 拉取，避免生产账号模式下受保护图片 401。
