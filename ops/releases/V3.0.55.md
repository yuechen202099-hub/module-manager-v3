## 需求描述

项目驾驶舱终端明细弹窗中的“安装人员”需要按该终端下资料组实际安装人员聚合展示，并显示每名安装人员的完成占比，不能继续只展示施工任务领取人或指派人。

## 修复内容

- `/local-test/tasks` 任务列表新增 `installer_distribution` 字段。
- PostgreSQL 后端按终端资料组聚合实际安装人员，优先取资料组 `installer`，再兜底 `constructor`、`creator`。
- 仅统计已有照片回流的资料组，完成占比按安装人员资料组数 / 终端已完成资料组数计算。
- JSON 本地状态路径同步生成 `installer_distribution`。
- 前端终端明细弹窗“安装人员”列改为展示资料组聚合结果，例如 `张三 60%、李四 40%`。
- 版本号同步为 V3.0.55。
