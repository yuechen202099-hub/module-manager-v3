# V3.0.45 生产发布记录

## 变更摘要

- 资料组条码扫描、图片准确率统计、人工复核清单统一限定为 4 张有效照片完整的资料组。
- 照片不足 4 张或多于 4 张的资料组不执行后台扫码，不计入准确率分母，也不进入无法识别/异常弹窗。
- 保留后台静默判断和每批 50 组的生产批处理原则，避免前台扫码和高 CPU 全量重算。
