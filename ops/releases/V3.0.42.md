## User Request

- 修复资料组条码“前面还有成功的，现在没成功”的算法问题。
- 人工复核弹窗明细保持分页加载，避免一次性加载过慢。

## Changed Scope

- 组级条码匹配 meter 时复用既有长扫码规则 `build_long_scan_match_key`，支持生产照片中的长表号条码匹配资料组表号匹配键。
- 旧照片级 `matched` 证据仍需命中当前资料组标准答案；已过期或串组旧值进入异常复核。
- 同一张旧 `matched` 照片中识别到多个码时，命中其他字段的码也会累计到组级判断；不命中的历史噪声不拉低结果。
- 条码复核弹窗继续分页请求，默认每页 20 组，只加载当前页照片缩略图。
