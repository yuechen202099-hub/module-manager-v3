## Change Type

- Small production feature, `+0.01`
- Scope: backend support for the construction WeChat miniprogram.

## Change Summary

- Added `/miniprogram/auth/bind` for first-time WeChat binding to an existing constructor account.
- Added `/miniprogram/auth/login` for one-click WeChat miniprogram login after binding.
- Added assigned construction task and group list APIs for the miniprogram field workflow.
- Added single-file upload bridging for `wx.uploadFile`, committing a full construction group batch after all staged files arrive.
- Added miniprogram activity endpoints that reuse the signed-in constructor identity.
