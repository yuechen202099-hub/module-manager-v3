## User Request

- 修复资料组条码准确率“之前还有成功，现在全部不成功”的算法问题。
- 人工复核弹窗明细改为分页加载，避免一次性加载太多照片导致驾驶舱变慢。

## Changed Scope

- 资料组条码聚合规则兼容旧照片级 `matched` 证据：照片已确认 matched 且 expected type 为表号、模块或采集器时，继续作为该资料组对应字段的已识别证据。
- 旧 `matched` 照片不会因为历史 normalized 值为空或口径变化被重新计入异常值。
- `mismatched` 照片仍保留为人工复核证据，异常不匹配不会被隐藏。
- 管理员驾驶舱条码复核弹窗改为分页请求，默认每页 20 组，只加载当前页照片缩略图。
