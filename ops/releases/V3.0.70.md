# V3.0.70 Production Release Record

## Change Type

- Bug fix and production UI cleanup, `+0.01`
- Scope: project board system-status removal, data-center photo loading reliability, and unconstructed-reset barcode cleanup.
- Version: `3.0.70`
- Date: 2026-06-29

## Change Summary

- Removed the unrelated system-status card from the project board.
- Stopped the project board from fetching `/local-test/system/status` for page rendering.
- Changed data-center group photo viewing to render native image object URLs instead of Element Plus image wrappers.
- Added preview-to-original fallback for data-center photo loading.
- Added per-photo load errors and stale async load guards so administrators do not see silent blank photo cards.
- Cleared collector, module, construction barcode, and manual barcode-pass evidence when administrators reset a group to unconstructed.
- Added a targeted regression verifier for the project-board/data-center photo behavior.

## Safety Notes

- No database migration is included.
- No production `.env`, `data`, `uploads`, OSS objects, or PostgreSQL data are modified by the release package.
- Data-center edits, audit logging, group search, barcode scans, and archive flows are unchanged.
- Asset-management board work remains in design; this release does not add asset tables or new asset data writes.
