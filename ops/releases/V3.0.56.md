## 问题描述

项目驾驶舱终端明细弹窗中的“安装人员”已改为资料组聚合，但生产数据中部分资料组记录的是施工账号，例如 `xa`。用户要求该字段与“安装人员完成量占比”卡片一致，展示账号管理中的姓名，而不是账号。

## 修复内容

- 安装人员聚合增加账号到姓名的转换，优先按账号表查找 `name`。
- PostgreSQL 任务列表的 `installer_distribution` 会把账号和姓名统一合并，例如 `xa` 与 `樊哲浩` 会合并展示为 `樊哲浩`。
- JSON 本地状态路径同步转换账号姓名，保持测试环境和生产环境口径一致。
- 保留 V3.0.55 的完成占比规则：只统计已有照片回流的资料组，占比按安装人员资料组数 / 终端已完成资料组数计算。
- 更新内容新增中文说明，版本号同步为 V3.0.56。
