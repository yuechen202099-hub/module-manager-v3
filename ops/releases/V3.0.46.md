# V3.0.46 生产发布记录

## 变更摘要

- 后台条码维护脚本新增 `--unprocessed-only`，只筛选 4 张有效照片完整且仍有照片未写入 `barcode_check_status` 的资料组。
- 后台条码维护脚本新增 `--auto-archive-passed`，扫描后资料组条码整体通过且照片已分类时，走既有批量归档链路自动归档。
- 后台条码维护脚本新增 `--archive-ready-only`，用于归档已经扫描通过但尚未归档的未审资料组。
- 新增 `scripts/run_photo_barcode_maintenance.sh`，每批 50 组、批间暂停、低优先级运行，并使用 `flock` 防止重复执行。
- 新增 systemd service/timer 模板，生产可每天 00:00 自动运行维护任务。

## 人工复核口径

- 只处理 4 张有效照片完整的资料组。
- 无法识别、异常不匹配、照片未分类、归档校验失败的资料组不自动归档，继续留在人工复核或异常流程中。
- 已经 `approved` 或 `rejected` 的资料组不重复自动归档。
