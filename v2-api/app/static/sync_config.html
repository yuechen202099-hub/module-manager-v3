供应商已明确不提供正式 API，所以系统不再引导输入 token 或请求 JSON。目前的正式工作流是：从导出的表格导入扫码数据，保存照片 URL 和关键字段，然后由审阅员领取终端任务并用快捷键完成分类归档。

1导出表格从供应商系统导出扫码表格，包含表号、采集器、模块、照片 URL 和安装人员。
2导入审阅工作台在审阅工作台上传 Excel/CSV，系统按表号和终端匹配资料组。
3领取并分类任务按终端领取，图片从 URL 加载，分类后由后端生成归档命名。
