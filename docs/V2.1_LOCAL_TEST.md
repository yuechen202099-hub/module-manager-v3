# V2.1 Local Test Build

## Goal

V2.1 local test build proves the core business workflow before cloud investment:

- Import the three provided Excel workbooks.
- Match stage rows to total catalog rows with the V1.3 barcode rule.
- Use total catalog as the only installation address source.
- Claim and release a review task.
- Review material groups locally.
- Save group status and notes in the in-memory simulation.
- Show live progress statistics.
