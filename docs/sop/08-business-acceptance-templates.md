# Business Acceptance Templates

## Admin Backoffice

- [ ] Admin can log in after daily session rules.
- [ ] Admin can search by meter, module, collector, terminal, installer.
- [ ] Admin group search shows installer and readonly photo thumbnails.
- [ ] Admin can edit allowed group fields.
- [ ] Edits create audit records.
- [ ] Admin can reset to unreviewed and unconstructed.
- [ ] Admin can bulk archive selected groups.

## Construction Page

- [ ] Construction page shows current version.
- [ ] Login expiration redirects to login before any operation.
- [ ] Scan unknown task shows no work order; no `00000000` work order appears.
- [ ] Offline cache does not create placeholder work orders.
- [ ] Automatic upload remains disabled unless explicitly re-enabled.
- [ ] Upload validates group/task relationship.

## Review Workflow

- [ ] Reviewer can claim task.
- [ ] Photo thumbnails load.
- [ ] Shortcut classification works.
- [ ] Classified photos archive with category filename.
- [ ] Reclassification and deletion reset review state as designed.
- [ ] Barcode photo checks are fail-safe.

## KPI And Dashboard

- [ ] Original KPI fields remain available.
- [ ] Online/KPI fusion respects attendance window and efficiency denominator rules.
- [ ] Dashboard shows `图片准确率`.
- [ ] Non-barcode images do not reduce image accuracy.
- [ ] Summary API includes photo accuracy fields.

## Release Notes

- [ ] Administrator can open update notes.
- [ ] Update notes are Chinese.
