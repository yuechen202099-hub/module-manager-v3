# 工作规则

规则版本：`Rules v1`

## 1. 角色和默认工作方式

Codex 是本项目默认总执行者。用户提出需求后，Codex 负责读取上下文、拆解任务、实施修改、运行验证、记录任务、必要时发布，并用中文汇报结果。

普通代码修复和小功能默认直接执行。以下动作必须先获得用户明确确认：

- 生产发布异常流程或手动生产操作。
- 数据库迁移、数据修复、生产数据改写。
- 删除文件或大范围重构。
- 修改权限、核心业务规则、主流程。
- 版本号 minor 或 major 升级。
- 影响 `.env`、`data`、`uploads`、OSS、PostgreSQL 的操作。

## 2. 生产保护底线

- 不覆盖生产 `.env`。
- 不覆盖生产 `data`。
- 不覆盖生产 `uploads`。
- 不直接修改 OSS 生产对象，除非用户单独确认。
- 不直接修改 PostgreSQL 生产数据，除非用户单独确认数据库方案、备份、dry-run 和回滚方案。
- 不在缺少备份、验证、回滚路径时发布生产。

## 3. 版本规则

- 当前应用基线：`V3.0.2`。
- 普通代码改动默认自动升 patch，例如 `V3.0.2` 到 `V3.0.3`。
- 大功能、数据库结构、架构、主流程变化先询问用户确认版本策略。
- 文档、协作规则、任务记录修改不升应用版本。
- 协作规则使用独立版本：`Rules v1`、`Rules v2`。

## 4. 任务记录

所有代码改动都要建立简短任务记录。当前基线路径：

```text
ops-v3/tasks/V3.0.2/
```

任务状态使用：

- 待处理
- 进行中
- 已验证
- 已发布
- 阻塞
- 取消

任务记录必须包含：

- 需求
- 原因
- 修改文件
- 影响范围
- 验证结果
- 风险/回滚

普通文档和协作规则改动可以记录在提交信息或规则文件中；不强制建立代码任务记录。

## 5. 验证规则

验证按风险选择：

- 小改：运行相关单元测试、脚本检查或静态检查。
- 后端核心流程：运行相关后端测试，必要时补充 API smoke check。
- 前端核心流程：运行前端构建和相关浏览器验证。
- 路由或 Vue 迁移相关：运行 `python scripts/verify_vue_migration_gate.py`。
- 发布前：运行相关测试、构建、生产健康检查。

可用脚本包括：

```powershell
python scripts/verify_vue_migration_gate.py
powershell -ExecutionPolicy Bypass -File .\scripts\build-vue-shell.ps1
powershell -ExecutionPolicy Bypass -File .\scripts\run-client-acceptance-gate.ps1
```

生产服务器健康检查脚本：

```bash
bash scripts/production_health_check.sh
```

如果某项验证因环境缺失无法执行，最终报告必须写清原因和剩余风险。

## 6. 自动发布规则

普通代码改动在以下条件全部满足后可自动发布：

- 相关测试通过。
- 构建通过。
- 生产健康检查脚本通过。
- 不涉及数据库迁移、数据修复、`.env`、`uploads`、OSS、PostgreSQL 或生产数据改写。

以下动作不得自动发布，必须单独确认：

- 数据库迁移。
- 数据修复。
- 修改生产 `.env`、`data`、`uploads`。
- 修改 OSS 生产对象。
- 修改 PostgreSQL 生产数据。
- 需要特殊回滚方案的发布。

## 7. 代码发现和编辑规则

- 改动前先读取 `AGENTS.md` 和相关文档。
- 推荐优先使用 codebase-memory-mcp 图谱工具；图谱不可用或结果不足时，直接使用 `rg` 和文件读取。
- 不回退用户已有改动，除非用户明确要求。
- 编辑范围保持贴近需求，不做无关重构。
- 优先沿用项目已有框架、样式、数据结构和脚本。

## 8. 前端和客户端规则

- Vue 是目标生产前端。
- 静态 HTML 只作为兼容入口，不新增生产静态 HTML 页面。
- UI 应保持工程工作台风格：密集但可读、层级清晰、操作高效、快捷键优先。
- 微信小程序和 APK 当前暂停主动维护；除非用户明确重新启用，否则不把它们作为验收阻塞项。

## 9. 最终报告格式

最终报告使用中文，包含：

```text
需求：
修改文件：
验证结果：
版本变化：
发布状态：
风险：
```

报告要简洁，但必须说明未验证项和剩余风险。
